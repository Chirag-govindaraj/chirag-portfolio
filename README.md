# Chirag Govindaraj Portfolio

A responsive, single-page industrial engineering portfolio built with plain HTML, CSS, and JavaScript.

## Run locally

Open `index.html` directly, or start a local server:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Personalize

- Professional details are tailored from Chirag's CV.
- Replace or expand experience content in `index.html` as needed.
- The downloadable CV is stored in `assets/Chirag-Govindaraj-CV.pdf`.
